Полный гайд по настройке 3X-UI на VPS

Панель 3X-UI — это современный форк классической 233-ui, переписанный на Go. Главное преимущество перед аналогами — поддержка самых современных протоколов обхода блокировок. Если вы решили купить VPS для 3X-UI, вам нужно знать несколько нюансов.

Почему выбирают протокол VLESS-Reality?

На текущий момент Reality — это самый надежный способ маскировки трафика. В отличие от стандартного TLS, Reality не имеет отпечатков (fingerprints), по которым системы глубокого анализа пакетов (DPI) могут вычислить прокси-сервер. Он "притворяется" трафиком к популярным сайтам (например, Microsoft или Google).

Плюсы Reality

  • • Невозможно заблокировать по протоколу
  • • Минимальный оверхед (скорость выше)
  • • Не требует регистрации домена

Требования к серверу

  • • ОС: Ubuntu 20.04 / 22.04
  • • Root доступ
  • • Процессор с поддержкой AES-NI

Установка в 3 шага

Мы рекомендуем использовать Docker-версию панели или скрипт автоматической установки. Вот команды для сервера One Dash:

# 1. Заходим по SSH и обновляем систему
sudo apt update && sudo apt upgrade -y

# 2. Устанавливаем необходимые утилиты
sudo apt install curl socat -y

# 3. Запускаем установщик 3X-UI
bash <(curl -Ls https://raw.githubusercontent.com/maciwi02/3x-ui/master/install.sh)

После завершения установки панель будет работать на порту 2053. Обязательно откройте порты в настройках фаервола вашего хостинг-провайдера (если он включен).

Часто задаваемые вопросы (FAQ)

Можно ли использовать один сервер на 5-10 человек? ▼
Да, 3X-UI позволяет создавать неограниченное количество пользователей (Inbounds). На тарифе Micro (1 Гб ОЗУ) вы сможете комфортно держать до 20 одновременных подключений без потери производительности.
Что такое BBR и нужно ли его включать? ▼
BBR (Bottleneck Bandwidth and RTT) — это алгоритм контроля перегрузки от Google. Он значительно увеличивает скорость передачи данных на "шумных" или дальних соединениях. На всех VPS от One Dash включение BBR доступно через стандартные настройки ядра Linux.
